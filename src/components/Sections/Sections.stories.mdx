# Hero

This mobile first Hero component is designed to responsively adopt to various
screen sizes. Layout of this component depends on the device width and has
media-query break for following thresholds:

- `28em` Tablets, iPads
- `46em` Small screens and Laptops
- `62em` Desktops and lage screens
- `73em` Extra large screens and TV
- `126em` UHD screens and beyond
